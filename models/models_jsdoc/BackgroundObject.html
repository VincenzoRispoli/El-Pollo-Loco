Class: BackgroundObject

BackgroundObject(imagePath, x)

This class is an extension of the MovableObject class and represents the Background objects.

Constructor

new BackgroundObject(imagePath, x)

The constructor of the BackgroudObjects class takes as an argument, the image path of each background object and the position x of each background object to be positioned. Finally, it calculates the y-axis position of each object by subtracting the height - 480 pixels.
Parameters:
Name Type Description
imagePath string the path of the image to be loaded
x number the position on the x-axis where the object will be positioned
