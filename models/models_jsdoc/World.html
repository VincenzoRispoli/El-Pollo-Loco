Class: World

World(canvas, keyboard)

This class represents the world where all objects and their animations are represented

Constructor

new World(canvas, keyboard)

This constructor of the World class takes as parameters the canvas, which will be used to call up the 2D context for drawing graphics and animations. Then it takes the keyboard object as a parameter and assigns it to the keyboard char, which will be used for the game commands. It then calls the draw() function to draw the elements within the context and the run() function to control events within the game
Parameters:
Name Type Description
canvas object the HTML element for the depiction of graphics and animations
keyboard object the object containing the game commands
