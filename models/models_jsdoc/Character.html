Class: Character

Character()

This class is an extension of the MovableObject class and represents the Character object.

Constructor

new Character()

The constructor of the Charcter class loads the initial image and the images of the alert, walk, wounding and death animations. Next, the applyGravity() function is called to simulate the force of gravity when the Character jumps. Then the animate() function is called to start the various animations and finally the youLost() function, which will only be activated if the Character has been defeated.
