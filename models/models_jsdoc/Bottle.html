Class: Bottle

Bottle()

This class is an extension of the MovableObject class and represents the Bottle object.

Constructor

new Bottle()

The constructor of the Bottle class loads the single image and the array of images for the animations. Then the animate() function is called for the various animations, and finally the playGrabbindSound() function is called, which contains a sound effect, which is only activated when a bottle is grabbed.
