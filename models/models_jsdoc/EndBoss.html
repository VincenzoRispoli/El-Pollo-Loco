Class: EndBoss

EndBoss()

This class is an extension of the MovableObject class and represents the End Boss object.

Constructor

new EndBoss()

The constructor of the End Boss class loads the initial image and the images of the alert, walk, wounding and death animations into the imageCache object. Then the animate() function is called to start the various animations and finally the gameOver() function, which will only be activated if the end boss has been defeated.
