This class is an instance of the MovableObject class and represents the Coins object.

Constructor

new Coins()

The constructor of the Coins class loads the single image and the array of images for the animation. Then the animate() function is called for the various animations, and finally the playSound() function is called, which contains a sound effect, which is only activated when a coin is picked up
