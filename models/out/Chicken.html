Class: Chicken

Chicken()

This class is an extension of the MovableObject class and represents the chicken object.

Constructor

new Chicken()

The constructor of the Chicken class loads the initial image and the walking image array of the chicken. It then sets a random value at dimension x ranging from 500 to 2000 pixels. Finally, it starts animations with the animate() function.
