Class: SmallChicken

SmallChicken()

This class is an extension of the MovableObject class and represents the small chicken object.

Constructor

new SmallChicken()

The constructor of the SmallChicken class loads the initial image and the walking image array of the small chicken. It then sets a random value at dimension x ranging from 500 to 2000 pixels and a random velocity ranging from 0.50 pixels to 1 pixel. Finally, it starts animations with the animate() function.
